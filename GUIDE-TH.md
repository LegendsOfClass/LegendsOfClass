# 🎮 คู่มือเอาเกมขึ้นเว็บ (ฉบับจับมือทำ ไม่ต้องมีความรู้คอมพิวเตอร์)

อ่านจบ ทำตามทีละข้อ ใช้เวลาประมาณ **20–30 นาที** ทำครั้งเดียว หลังจากนั้นคุณจะได้
**ลิงก์เว็บเกมของตัวเอง** เปิดเล่นจากมือถือหรือคอมก็ได้ ส่งให้เพื่อนเล่นด้วยกันได้

> 💰 **ค่าใช้จ่าย: 0 บาท** และ**ไม่ต้องใช้บัตรเครดิต**เลยสักขั้นตอน

---

## ภาพรวม: เราจะสมัคร 3 เว็บ (ใช้อีเมลเดียวกันได้หมด)

| เว็บ | เปรียบเหมือน | หน้าที่ |
|---|---|---|
| **GitHub** | ตู้เก็บแบบแปลนบ้าน | เก็บไฟล์เกมของเรา |
| **Neon** | ตู้เซฟ | เก็บข้อมูลผู้เล่น ID รหัสผ่าน เลเวล ไอเทม |
| **Render** | ที่ดิน + เครื่องปั่นไฟ | รันเกมให้ออนไลน์ 24 ชม. แจกลิงก์ให้เรา |

ลำดับการทำ: เตรียมไฟล์ → GitHub → Neon → Render → ได้ลิงก์ 🎉

---

## ขั้นที่ 0: เตรียมไฟล์ (2 นาที)

1. ดาวน์โหลดไฟล์ **`legends-game-upload.zip`** ที่ผมส่งให้ในแชท
2. หาไฟล์ที่ดาวน์โหลดมา (ปกติอยู่ในโฟลเดอร์ **Downloads**)
3. **คลิกขวา** ที่ไฟล์ → เลือก **Extract All...** (แตกไฟล์ทั้งหมด) → กด **Extract**
4. จะได้โฟลเดอร์ชื่อ `legends-game-upload` — **เปิดเข้าไปข้างใน** จะเห็นไฟล์และโฟลเดอร์หลายอัน เช่น `packages`, `database`, `package.json`
5. เปิดโฟลเดอร์นี้ค้างไว้ เดี๋ยวขั้นที่ 1 จะใช้

---

## ขั้นที่ 1: GitHub — ฝากไฟล์เกม (10 นาที)

### 1.1 สมัครบัญชี
1. เข้า **https://github.com/signup**
2. กรอกอีเมล → ตั้งรหัสผ่าน → ตั้งชื่อผู้ใช้ (ภาษาอังกฤษ) → ทำตามที่เว็บถามจนจบ
3. เช็คอีเมลเพื่อกดยืนยัน (ถ้ามี)

### 1.2 สร้าง "กล่อง" เก็บไฟล์ (เรียกว่า Repository)
1. ล็อกอินแล้ว มองหาปุ่ม **+** ที่**มุมขวาบน** ของเว็บ → กด → เลือก **New repository**
2. ช่อง **Repository name** พิมพ์: `legends-game`
3. เลือก **Public** (จำเป็น เพื่อให้ Render ฟรีมองเห็น)
4. **ไม่ต้องติ๊ก**อย่างอื่นใดๆ → กดปุ่มเขียว **Create repository**

### 1.3 อัปโหลดไฟล์เกม ⭐ ขั้นตอนสำคัญที่สุด
1. ในหน้าที่เปิดขึ้นมา จะมีลิงก์เล็กๆ เขียนว่า **uploading an existing file** → กดตรงนั้น
2. เปิดโฟลเดอร์ `legends-game-upload` ที่แตกไว้ (จากขั้นที่ 0)
3. ⚠️ **สำคัญมาก:** กด **Ctrl + A** (เลือกทุกอย่าง**ข้างใน**โฟลเดอร์) แล้ว**ลากทั้งหมด**ไปวางในกรอบกลางหน้าเว็บ GitHub
   - ✅ ถูก: ลาก "ของข้างใน" ทั้งหมด (packages, database, package.json, ...)
   - ❌ ผิด: ลากตัวโฟลเดอร์แม่ `legends-game-upload` ไปทั้งก้อน
4. รอแถบอัปโหลดวิ่งจนครบ (~63 ไฟล์)
5. เลื่อนลงล่างสุด → กดปุ่มเขียว **Commit changes**
6. เสร็จแล้วหน้ากล่องต้องเห็นโฟลเดอร์ `packages`, `database` และไฟล์ `render.yaml` — ถ้าเห็นครบ ✅ ผ่าน!

## ขั้นที่ 2: Neon — สร้างตู้เซฟข้อมูลผู้เล่น (5 นาที)

1. เข้า **https://neon.tech** → กด **Sign Up**
2. เลือก **Continue with Google** (ง่ายสุด) หรือสมัครด้วยอีเมลก็ได้
3. เว็บจะพาสร้างโปรเจกต์แรก (Create project):
   - **Project name**: พิมพ์ `legends-game`
   - **Region**: เลือก **Asia Pacific (Singapore)** ← ใกล้ไทยที่สุด เกมจะลื่น
   - อย่างอื่นปล่อยตามเดิม → กด **Create**
4. เสร็จแล้วมองหาปุ่ม/กล่อง **Connect** หรือ **Connection string**
5. จะเห็นข้อความยาวๆ หน้าตาประมาณนี้:
   ```
   postgresql://neondb_owner:npg_xxxx@ep-xxxx-pooler.ap-southeast-1.aws.neon.tech/neondb?sslmode=require
   ```
6. กดปุ่ม **Copy** 📋 แล้ว**วางเก็บไว้ใน Notepad ก่อน** — เดี๋ยวขั้นที่ 3 ต้องใช้
   > 🔒 ข้อความนี้คือ "กุญแจตู้เซฟ" **ห้ามโพสต์สาธารณะหรือส่งให้คนอื่น**เด็ดขาด

---

## ขั้นที่ 3: Render — เปิดเกมออนไลน์ (10 นาที)

1. เข้า **https://render.com** → กด **Get Started** / **Sign Up**
2. เลือก **Sign up with GitHub** ← สำคัญ! จะได้เชื่อมกับไฟล์เกมอัตโนมัติ
3. GitHub จะถามว่าอนุญาตไหม → กด **Authorize**
4. เข้ามาหน้าหลัก (Dashboard) → กดปุ่ม **New +** (มุมขวาบน) → เลือก **Blueprint**
5. จะเห็นรายชื่อกล่อง GitHub ของเรา → หา **legends-game** → กด **Connect**
6. Render จะอ่านพิมพ์เขียว (`render.yaml`) แล้วรู้เองว่าต้องสร้างอะไร จะมีช่องให้กรอก:
   - **DATABASE_URL** → **วางกุญแจจาก Neon** ที่เก็บไว้ใน Notepad
   - (ค่าอื่นๆ Render เติมให้เองหมด)
7. กดปุ่ม **Apply** / **Deploy Blueprint**
8. ☕ รอ 5–10 นาที — จะเห็นตัวหนังสือวิ่งๆ (เว็บกำลัง "ก่อสร้าง") รอจนขึ้นสถานะ **Live** สีเขียว
9. ที่ด้านบนของหน้า จะมีลิงก์หน้าตาแบบนี้:

   ### 🔗 `https://legends-of-class-evolution-xxxx.onrender.com`

   **นี่แหละลิงก์เกมของคุณ!** กดเปิด → สมัคร ID → เล่นได้เลย 🎉

---

## ✅ ทดสอบว่าสำเร็จ

1. เปิดลิงก์เกม → เห็นหน้า Login
2. กด "สมัครสมาชิก" → ตั้ง ID / รหัสผ่าน (8 ตัวขึ้นไป) / ชื่อตัวละคร
3. เข้าเกม → เห็นเมือง เดินได้ เปลี่ยนอาชีพได้ ออกไปตีสไลม์ได้
4. **ปิดเบราว์เซอร์ แล้วเปิดลิงก์ใหม่ → ล็อกอิน → ของทุกอย่างยังอยู่ = เซฟออนไลน์ทำงาน!**
5. ส่งลิงก์ให้เพื่อนสมัคร ID ของตัวเองแล้วเล่นพร้อมกันได้เลย

## 📌 เรื่องควรรู้ (สำคัญ อ่านหน่อยนะ)

**1. เว็บฟรีจะ "หลับ" เมื่อไม่มีคนเล่น 15 นาที**
คนแรกที่เปิดลิงก์หลังเว็บหลับ ต้องรอประมาณ **1 นาที** (หน้าจอจะค้างๆ ขาวๆ) — **ปกติ ไม่ได้พัง!** พอเว็บตื่นแล้วทุกคนจะเข้าเร็วตามปกติ ถ้าอยากหายต้องอัปเกรดเป็นแบบเสียเงินภายหลัง (ยังไม่จำเป็นตอนทดสอบ)

**2. อัปเดตเกมรอบหน้า ง่ายกว่าเดิมมาก**
Milestone ต่อไปเสร็จ ผมจะส่ง zip ใหม่ให้ → คุณอัปโหลดไฟล์ทับใน GitHub (ขั้น 1.3 เดิม) → Render สร้างใหม่ให้**อัตโนมัติ** → **ลิงก์เดิมตลอด** เพื่อนไม่ต้องจำลิงก์ใหม่ และเซฟทุกคนอยู่ครบเพราะข้อมูลอยู่ที่ Neon

**3. เวอร์ชันออนไลน์ไม่มีปุ่มโกง (Dev Tools)**
ระบบปิดเครื่องมือนักพัฒนาบนเว็บจริงให้อัตโนมัติ เพื่อความแฟร์ — เครื่องมือพวกนั้นมีไว้ตอนผมพัฒนาเท่านั้น

**4. ขีดจำกัดของฟรี**
เหมาะกับช่วงทดสอบ + เพื่อนไม่กี่สิบคน ถ้าเกมดังมีคนเล่นเยอะจริงค่อยขยับขึ้นแผนเสียเงิน (เดือนละไม่กี่ร้อยบาท) โดยไม่ต้องแก้อะไรเลย

## 🆘 ถ้าติดปัญหา

| อาการ | วิธีแก้ |
|---|---|
| Render ขึ้น **Build failed** สีแดง | กดเข้าไปดูหน้า **Logs** ถ่ายรูปหน้าจอส่งให้ผมในแชท เดี๋ยวผมบอกให้ว่าแก้ตรงไหน |
| ลืมวาง DATABASE_URL หรือวางผิด | ในหน้า service ที่ Render → เมนู **Environment** → แก้ค่า `DATABASE_URL` → กด Save (เว็บจะรีสตาร์ทเอง) |
| เปิดลิงก์แล้วขาวนานเกิน 2 นาที | รีเฟรช 1 ครั้ง ถ้ายังไม่มา ดู Logs แล้วถ่ายรูปส่งผม |
| สมัคร ID ในเกมไม่ผ่าน | ID ต้องเป็นอังกฤษ/ตัวเลข 3 ตัวขึ้นไป รหัสผ่าน 8 ตัวขึ้นไป และชื่อตัวละครห้ามซ้ำกับคนอื่น |
| อัปโหลด GitHub แล้วไฟล์ไม่ครบ | เข้ากล่อง → ปุ่ม **Add file → Upload files** → ลากไฟล์ที่ขาดเพิ่มได้เสมอ |

ติดขั้นไหน **ถ่ายรูปหน้าจอมาถามผมได้ทุกขั้นตอน** ผมจะดูให้ว่าอยู่ตรงไหนและต้องกดอะไรต่อครับ 😊
